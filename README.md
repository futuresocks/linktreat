# Linktreat

A command-line tool to generate your own 'link' 'tree' website.

## Usage

```
> generate-linktreat conifg.json dist
Generating linktreat...
```

## Configuration

```json
{
  "profilePicture": "./assets/pic.png",
  "name": "Futuresocks",
  "links": [{ "title": "Github", "url": "https://github.com/futuresocks" }]
}
```
